## 🌿 베란다집사

'베란다 집사'는 스마트한 식물 관리 애플리케이션으로, 사용자가 원거리에서 식물의 상태를 모니터링하고 관리할 수 있도록 설계되었다.

## ✅ 핵심 기술
1. 식물 AR
    - 식물 종류 구분
    - 식물 실시간 모니터링
      - 병충해 감지
      - 온습도 값 제공
2. 병충해 진단 및 진단서 제공
    - 병충해 사진 촬영하여 업로드할 경우 병충해 진단
    - 해당 병충해에 대한 진단서 제공
3. 생장환경 자동화
    - 일정 토양 습도값 미만이 되면 물펌프 자동으로 작동
    - 대기의 온도, 습도 측정
